# DyT Composition Study

Code and artifact repository for:

**When Does Removing LayerNorm Help? Activation Bounding as a Regime-Dependent Implicit Regularizer**

This repository is being staged for public release. The initial public release will be synchronized with the arXiv record and will include:

- training code and configuration files for the GPT-2-family, Llama-style, and ViT experiments;
- analysis scripts for significance tests, saturation measurement, predictor validation, and downstream evaluations;
- machine-readable result JSONs used to generate the paper tables and figures;
- documentation for reproducing the main experiments and validating artifact provenance.

Large model checkpoints and raw datasets are not stored directly in Git. Public dataset/checkpoint pointers will be added after the arXiv page is live.
